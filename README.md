# MiniGrameForRelax

> Game for relax

## Build setup

### [yarn](https://yarnpkg.com) - recommend
``` bash
# Install dependencies
yarn install

# Server with hot reload stylesheet and javascript
yarn run dev
```

### [npm](https://www.npmjs.com/)
``` bash
# Install dependencies
npm install

# Server with hot reload stylesheet and javascript
npm run dev
```
